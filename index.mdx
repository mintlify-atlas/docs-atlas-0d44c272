## Why Mercury?

Mercury goes beyond typical AI chatbots with a set of design principles that keep you in control.

<CardGroup cols={2}>
  <Card title="Permission-hardened" icon="shield-halved" href="/features/permissions">
    Folder-level read/write scoping, shell blocklist, and inline approval flows. No silent actions.
  </Card>
  <Card title="Soul-driven" icon="heart" href="/introduction">
    Personality defined by markdown files you own and can edit: `soul.md`, `persona.md`, `taste.md`.
  </Card>
  <Card title="Token-aware" icon="coins" href="/features/token-budget">

## Get running in seconds

<Steps>
  <Step title="Install Mercury">
    Run Mercury directly with `npx`, or install it globally:

    <CodeGroup>
    ```bash npx (no install)
    npx @cosmicstack/mercury-agent
    ```

    ```bash Global install
    npm i -g @cosmicstack/mercury-agent
    mercury
    ```
    </CodeGroup>
  </Step>
  <Step title="Complete the setup wizard">
    On first run, Mercury asks for your name, an LLM API key (DeepSeek is free to start), and optionally a Telegram bot token. Takes 30 seconds.
  </Step>
  <Step title="Start chatting">
    Type your first message and press Enter. Mercury uses its built-in tools to help you with tasks — and asks permission before taking any action it hasn't been authorized for.
  </Step>
  <Step title="Run 24/7 (optional)">
    Run `mercury up` to install Mercury as a system service and start it in the background. Telegram becomes your primary interface — chat from anywhere.
  </Step>
</Steps>

## Supported LLM providers

Mercury supports multiple LLM providers with automatic fallback. Configure one or several:

| Provider | Default Model | Notes |
|----------|--------------|-------|
| **DeepSeek** | deepseek-chat | Default, cost-effective |
| **OpenAI** | gpt-4o-mini | GPT-4o, o3, and more |
| **Anthropic** | claude-sonnet-4-20250514 | Claude Sonnet, Haiku, Opus |
| **Grok (xAI)** | grok-4 | OpenAI-compatible |
| **Ollama Cloud** | gpt-oss:120b | Remote Ollama via API |
| **Ollama Local** | gpt-oss:20b | No API key needed |

<Tip>
  Start with DeepSeek — it's cost-effective and works great out of the box. You can add more providers later with `mercury doctor`.
</Tip>
